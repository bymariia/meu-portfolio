Sobre mim

Olá, eu sou Maria Luiza, uma desenvolvedora Full-Stack e analista de dados em formação. Tenho 18 anos e sou estudante de Ciência da Computação na Atitus Educação. Atualmente, sou bolsista no programa AWS Cloud Data Engineer da Compass UOL, onde aprofundo meus conhecimentos em soluções de dados em nuvem. Minha paixão pela tecnologia me levou a explorar diversas áreas, desde o desenvolvimento web (HTML, CSS, JavaScript) até a análise e visualização de dados com Python (Pandas e Matplotlib) e gerenciamento de banco de dados (SQL). Este portfólio é um reflexo do meu aprendizado e dedicação contínua. Acredito que a combinação de criatividade, lógica e uma abordagem proativa é fundamental para construir soluções inovadoras. Estou sempre em busca de novos desafios para aplicar meus conhecimentos e evoluir como profissional.
